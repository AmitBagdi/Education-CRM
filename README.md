# Education CRM System (Learnifly Education)

A **full-stack Education CRM web application** built using **Spring Boot** to manage courses, users, employees, inquiries, orders, and feedback for an education platform.

This project helps educational institutes efficiently handle **course sales, customer inquiries, follow-ups, and employee management** through a centralized system.

# Features

# Admin Module
- Admin login
- Course management (Add / Update / Delete)
- Employee management
- View sales reports & charts
- Manage customers & feedback

# Employee Module
- Employee login
- View assigned inquiries
- Follow-ups management
- Sales tracking

# User Module
- User registration & login
- View available courses
- Buy courses
- View purchased courses
- Submit feedback

# CRM Functionalities
- Inquiry management
- Follow-up tracking
- Order & sales management
- Feedback system
- Dashboard with charts
